## Database Structure (PostgreSQL Concept)

### Database
- A database is like a big container that stores all data
- One PostgreSQL server can contain multiple databases
- Each database is isolated from others

## Schema
- Schema is used to organize database objects
- Think of a schema as a folder
- Helps organize tables, views, functions, etc.
- By default, PostgreSQL has a schema called `public`


## Table
- A table stores data
- Structure is similar to an Excel sheet
- Consists of:
  - Rows - Records
  - Columns - Attributes
