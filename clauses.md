## clauses
they are mainly used for querying the data and are kind of building blocks 
which we use to extract the data.
1. select- choose which column to display
2. from- specify the table
3. where- filter names based on condition
4. group by- group rows for aggregation
5. having- filter aggregated groups (used after group by)
6. order by- sort result in ascending or descending order
7. as- aliasing columns or table temporarily
8. distinct- return only unique distint values
