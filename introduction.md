## What is a Database?

A database is a collection of organized data that can be easily
accessed, managed, and updated.

### Key Points
- Stores data electronically
- Data is organized in tables
- Allows easy access, modification, and deletion of data
- Data can be queried using a language (usually SQL)

### Advantages of a Database
- Ensures data consistency
- Improves data reliability
- Provides data security
- Makes data management efficient

## RDBMS (Relational Database Management System)

RDBMS stores data in the form of tables, where tables are related
to each other.

### Breakdown of RDBMS
- Data - Information stored in tables
- Database-Collection of structured data
- Management- Handling and controlling data
- System - Software that manages the database

RDBMS allows users to create, read, update, and delete data
using SQL.



## SQL (Structured Query Language)

SQL is a language used to interact with databases.

### Uses of SQL
- Read data
- Insert data
- Update data
- Delete data

### Examples of SQL-based Databases
- PostgreSQL
- MySQL
- SQLite
- SQL Server
